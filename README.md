# Petra - Portfolio Web

## Español

Sitio web portfolio de **Petra**, artista de nuevos medios cuya obra se sitúa en el cruce entre danza contemporánea, arte digital y tecnologías emergentes.

### Tecnologías
- HTML5, CSS3, JavaScript
- Bootstrap 5
- Swiper.js
- Anime.js
- Vimeo/YouTube embeds

### Estructura
```
├── index.html          # Página principal
├── about/              # Biografía
├── articulos/          # Artículos
├── colecciones/        # Colecciones de obra
├── cryptoart/          # CryptoArt
├── css/                # Estilos
├── exhibiciones/       # Exhibiciones
├── imagenesOBRA/       # Imágenes de obras
├── imagesNEWS/         # Imágenes de noticias
├── js/                 # Scripts
├── noticias/           # Noticias
├── talleres/           # Talleres
└── shaders/            # Shaders GLSL
```
